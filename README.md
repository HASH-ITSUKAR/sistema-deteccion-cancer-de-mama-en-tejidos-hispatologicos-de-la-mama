# sistema-deteccion-cancer-de-mama-en-tejidos-hispatologicos-de-la-mama
Sistemas de deteccion para la deteccion de cancer de mama en tejidos histológicos de la mama. Software para obtener el grado de ing de sistemas en la UNT

MODELOS TRANSFORMERS:
Inception, DenseNet, EfficientNet
MODELOS CNN:
CaiT, DeiT, PVT, Swin, ViT
MODELO HIBRIDO POR STACKING:
XGBoost

Link de descarga de los modelos:
https://drive.google.com/drive/folders/14wE8YLrUcNVnoUkeTD9KxVti1iu9sjgn?usp=sharing

Modo instalacion de los modelos:
1. Entrar al backend
2. Entrar a la carpeta llamada modelo
3. Entrar a la carpeta llamada modelos
4. Descargar los modelos y poner todos los modelos en esa carpeta
